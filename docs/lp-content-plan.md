# SwimHub LP コンテンツ計画

## 1. 練習のタイムを記録

- サークル・本数・セット数を記載して、回った記録をつける
  - 次回はもう1本多く回る、サークルを5秒短くして挑戦するなど
- アベレージタイムを記載
  - 次回はそれを目標タイムにして挑戦
- タグをつけることで一覧で見ることができる
  - 例: `High-average` や `lactate` といったタグをつけて、後でフィルタリングして確認可能

## 2. 大会のタイムを記録

- 当日の自分のタイムを記録しておく
  - 途中タイムからラップタイムは自動で計算される
  - それに合わせて自分のベストタイム表も自動更新される
  - ベストタイム表は短水路/長水路、引き継ぎあり/なしまでデータに入る
- チームで利用時はエントリーも管理可能

## 3. コーチやマネージャーが代理で入力

- 練習記録や大会記録は、選手自身が入力できるのはもちろん、コーチやマネージャーがチームメンバーの練習記録や大会記録を一括で登録することが可能
- 記録はあくまでも個人に紐付くので、そのチームを引退した後も記録は蓄積される
  - **一生使えるアプリ**

## 4. 目標管理が可能

- どの試合でどの種目に出て何秒出すのかを一目でわかるようにしておく
- それを踏まえて練習では、何秒アベレージで出すのか、どのくらいのサークルを何本回れるようになるかなど、マイルストーンで置くことができる
  - 練習記録はトラッキング可能なので、マイルストーンは自動で進捗が更新される

## イチオシ便利機能

- 手書きの練習記録画像をAIで解析することで入力の手間を省ける
  - これが便利なので **SwimHub Scanner** というアプリも個別にリリース
